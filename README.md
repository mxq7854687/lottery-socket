### Project Setup
setup venv (if needed) 
```
python3 -m venv
activate vevn/bin/activate
pip3 install requirements.txt
#deactivate venv 
deactivate
```
## How to run
```
# server start at http://localhost:5000
python3 server.py
```

    
    
### Run test case
```
python3 test.py
```
